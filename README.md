# memo
이 프로젝트는 프로그래머스의 라인 안드로이드 앱 챌린지에 참가해서 진행했던 메모장 앱이다.

해당 앱의 기능은
- 텍스트 메모 저장, 삭제, 수정
- 텍스트 메모에 이미지 추가
  - 이때 이미지는 갤러리에서 가져오거나, 카메라로 찍어서 첨부하거나, URL 경로를 통해 첨부할 수 있다. (하지만 해당 앱은 갤러리만 구현됨)
  - 이미지는 다중 첨부가 가능하다.
  - 첨부된 이미지 중 제일 앞의 이미지를 메모 리스트에서 썸네일로 보여줌.
